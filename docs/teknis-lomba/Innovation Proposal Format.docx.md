

**GUIDELINES FOR DESCRIPTION OF WORK**   
**(INNOVATION PROPOSAL)**

1. Description of Work is written in Bahasa.  
2. Description of Work should be a maximum of 1000 words.  
3. Description of Work should use headers as provided in the example.  
4. Please refer to the attached formatting example for reference.

**NOTES**

1. Description of Work is written without any identifying information, including the author's name and university/school affiliation.  
2. The work description must be submitted in PDF format, with a maximum size of 10 MB, and submitted in the following format:   
   Description\_Team Name\_Sub Theme\_Project Title.pdf. 

   *ex: Description\_Freeport Team\_Health and Safety\_Usage of software-based anti-broken rock helmets.*

Theme  
Subtheme  
**Work Title**

Contains an explanation about the work that your team is going to create, including its purposes, underlying background, problems and how your work will be able to solve it, and the logic/way of work, but are not limited to only the things mentioned, you are free to add any parts or explanation. Description of work is written in proper and correct English or Bahasa. The description of work is limited to a maximum of 1000 words and should be justified (aligned to both left and right margins).
